# SOC AI Reference Website

A lightweight static website designed for GitHub Pages. It complements a talk about practical AI use in a SOC environment and keeps all examples public-safe and sanitized.

## Structure

- `index.html` - landing page and quick map
- `prompts.html` - SOC prompt patterns and copy-ready examples
- `capabilities.html` - practical AI use cases by workflow
- `safety.html` - sanitization, validation, and attacker use guidance
- `resources.html` - curated public links, repos, and official references
- `assets/css/styles.css` - shared theme and component styles
- `assets/js/site.js` - mobile nav, prompt copy, and resource filters
- `assets/img/` - decorative assets

## Local preview

Open `index.html` directly in a browser for a quick review, or serve the folder with any static file server if you want behavior closer to GitHub Pages.

## GitHub Pages notes

- This site uses only relative links, so it works well on GitHub Pages project hosting.
- No build step is required.
- To publish, push the files to a repository and enable GitHub Pages for the branch/folder you want to host.

## Design

The visual language follows a warm editorial style:

- ivory and parchment backgrounds
- terracotta and clay accents
- serif headings with clean sans-serif body text
- spacious layouts and concise content blocks
